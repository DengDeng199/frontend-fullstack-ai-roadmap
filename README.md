## 📋 项目简介

本仓库是一份以**产出驱动**为核心的个人技术成长体系，聚焦于将 Vue 前端工程师升级为：

> **Vue 前端专家 + Node.js BFF 全栈 + AI 工具高手 + 民航/政务行业理解**

这四项叠加，在当前中国市场是稀缺复合型能力组合。

- [MyBatis 官方文档](https://mybatis.org)

### AI 赋能
- [Prompt Engineering Guide](https://promptingguide.ai)
- [Dify 官方文档](https://dify.ai)
- LangChain.js 文档

## 💡 学习节奏建议

- **每月只主攻 1-2 个方向**，其他保持低频接触
- 所有学习最终要在真实项目里落地（caidiaweb / odvp-web 等）
- **产出驱动**：每个阶段都有具体产出目标，不只看视频/看书
- 每周至少 10 小时有效学习时间（工作日每天 1h + 周末 5h）
