## 四、学习节奏建议

- **每月只主攻 1-2 个方向**，其他保持低频接触
- 所有学习最终要在真实项目（caidiaweb / odvp-web / bcs-web）里落地
- **产出驱动**：每个阶段都有具体产出目标，不只看视频/看书
- 每周至少 10 小时有效学习时间（工作日每天 1h + 周末 5h）
