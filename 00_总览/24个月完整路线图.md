# 前端工程师复合成长规划 · 24 个月路线图

> 目标：前端专家 + Node.js BFF 全栈 + AI 工具赋能 + Java 轻量了解
> 定位：成为公司内不可替代的复合型工程师

---

## 能力模型总览

| 方向 | 目标层级 | 定位 | 预计周期 |
|------|---------|------|---------|
| 前端专家 | L3 精通 | 核心竞争力 | 4-5 个月 |
| Node.js BFF | L3 精通 | 全栈扩展 | 5-6 个月 |
| AI 工具赋能 | L2-L3 | 效率杠杆 | 贯穿全程 |
| Java 后端 | L1 了解 | 协作桥梁 | 3-4 个月低频 |

## 方向一：成长为前端专家（L3）

### 第 1 阶段：JS 语言深度（2 周）
- 闭包原理 / 原型链 / this 绑定规则
- Event Loop 事件循环机制
- Promise / async-await 底层实现
- WeakMap / Proxy 元编程
- **产出**：能默写 Promise.all / 防抖节流 / 深拷贝底层实现

### 第 2 阶段：浏览器原理（3 周）
- 渲染管线：HTML 解析 → CSSOM → Layout → Paint → Composite
- 重排 vs 重绘触发条件
- 虚拟 DOM diff 算法本质
- V8 垃圾回收机制
- **产出**：能排查首页 FCP/LCP 慢的原因并给出优化方案

### 第 3 阶段：性能监控体系（3 周）
- Core Web Vitals 指标体系
- Performance API / Long Task 监控
- 内存泄漏排查 / 代码分割策略
- 图片懒加载与骨架屏
- **产出**：在当前项目搭建前端性能监控面板，输出优化报告

### 第 4 阶段：工程化深度（4 周）
- Vite 构建原理与插件开发
- ESLint 规则自定义 / Turbo monorepo
- 自动化测试：Vitest 单元 + Cypress E2E
- CI 流水线配置
- **产出**：为团队写一个 Vite 插件 + 覆盖核心模块单测到 80%

### 第 5 阶段：可视化专项（4 周）
- ECharts 深度：自定义系列 / 大数据量渲染优化 / 地图定制
- Canvas 基础 / WebGL 入门
- 数据可视化设计原则
- **产出**：做一个行业相关的可视化 Demo（如电磁态势大屏）

### 第 6 阶段：架构与设计系统（4 周）
- 组件设计模式：Renderless / HOC / Composable / 控制反转
- 微前端原理（qiankun）/ 设计系统搭建
- 技术方案文档撰写
- **产出**：主导一个模块的技术选型方案，输出架构设计文档

**推荐资源**：
- 书籍：《你不知道的 JS》《JS 高级程序设计》《Web 性能权威指南》
- 课程：极客时间《前端进阶》、掘金小册《Vite 原理与插件开发》
- 实践：Chrome DevTools Performance 面板反复练习

## 方向二：Node.js BFF 全栈（L3）

### 第 1 阶段：Node.js 核心（3 周）
- 模块系统（CJS / ESM）/ fs / path / http 核心模块
- Event Loop 机制（与浏览器差异）/ Stream 流处理 / Buffer
- **产出**：用原生 http 模块写一个静态文件服务器

### 第 2 阶段：Nest.js 框架（4 周）
- 装饰器 / 依赖注入 / 模块化架构
- Controller-Service-Repository 分层
- Guards / Interceptors / Pipes / Filters 请求生命周期
- **产出**：搭建 Nest.js 项目骨架，实现 CRUD 接口

### 第 3 阶段：数据库 + ORM（4 周）
- MySQL 索引优化 / 事务隔离级别 / 慢查询分析
- Prisma ORM 或 TypeORM / Redis 缓存策略
- **产出**：实现带分页、搜索、关联查询的 RESTful API

### 第 4 阶段：BFF 模式实战（3 周）
- BFF 层定位：接口聚合 / 数据裁剪 / 协议转换
- SSR 服务端渲染（Nuxt.js）/ API Gateway 模式
- 鉴权中间件
- **产出**：为公司项目搭建 BFF 层，聚合 Java 后端接口

### 第 5 阶段：部署与运维（3 周）
- Docker 镜像构建 / docker-compose 编排
- Nginx 反向代理配置 / PM2 进程管理
- 日志收集 / 健康检查
- **产出**：把 BFF 服务 Docker 化并部署到公司服务器

### 第 6 阶段：进阶 — 消息队列 + 实时（3 周）
- RabbitMQ / Bull Queue 消息队列
- Cron 定时任务 / WebSocket 实时推送
- 文件上传下载
- **产出**：实现一个 WebSocket 实时通知功能

**推荐资源**：
- 官方文档：Nest.js 官方文档（写得非常好，直接看就够）
- 课程：掘金小册《Nest.js 通关秘籍》、B 站《Node.js 从入门到实战》

## 方向三：Java 后端（L1 轻量了解）

> 定位：不写 Java 业务代码，但要能看懂、能改配置、能和后端同学高效沟通

### J1 · Java 语法速通（2 周）
- 强类型 / 接口 vs 类型 / 泛型 vs 联合类型
- 异常处理机制 / Stream API
- **方法**：与 JS 对比学，只学差异部分

### J2 · 能读 Spring Boot 项目（3 周）
- 理解 Controller / Service / Mapper 分层
- @Autowired 依赖注入 / 配置文件

### J3 · 能看 MyBatis SQL 映射（2 周）
- XML 映射文件 / 动态 SQL / 结果映射
- 一对多关联查询

### J4 · 协作桥梁能力（持续积累）
- 看懂 Swagger 文档
- 理解 JWT 鉴权流程
- 能定位后端报错日志

**核心产出**：
1. 接口联调时能自己看懂 Java 代码定位问题
2. 提需求时用后端听得懂的语言描述
3. 能接手简单的 Java 配置修改

## 方向四：AI 工具赋能（贯穿全程）

> 定位：不是转行 AI，而是让 AI 成为你最强大的杠杆工具

### A1 · AI 编码工具精通（立即开始 · 2 周上手）
- Cursor 全流程使用
- GitHub Copilot / Windsurf / Claude Code

### A3 · AI 工作流搭建（第 4-9 个月）
- Dify 搭建知识库问答
- n8n 自动化流程 / LangChain 基础
- RAG 原理

### A4 · AI 前端集成开发（第 6-12 个月）
- 流式输出（SSE）/ Markdown 渲染
- AI 对话组件 / 代码高亮

### A5 · AI 驱动的工程效率（持续优化）
- AI Code Review 自动化
- 自动生成单测 / 文档自动生成
- 需求文档到代码

**核心产出**：日常编码效率提升 3x + 为团队搭建 1 个 AI 辅助工具 + 主导 AI 工具落地推广

## 核心原则

1. **不要等精通再动手** — 边用边学，在真实项目中产出价值
2. **每个阶段的"产出目标"比"学了什么"更重要** — 让别人看见你的成长
3. **AI 工具从第一天就开始用** — 它是你提效的杠杆，不是额外负担
4. **Java 只需了解级** — 你的目标不是替代后端，而是成为桥梁
5. **所有学习最终服务于公司业务** — 在 caidiaweb / odvp-web / bcs-web 中落地
